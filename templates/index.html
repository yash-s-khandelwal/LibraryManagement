LibraryManagement Suite

To use the application, set up a virtual environment with Flask and flask-sqlalchemy installed

Next launch the server by running the app.py file

* to add a record, press the "Add a Book" button on the navbar, enter the necessary fields and click the submit button

* to see the list of all records, click on the "Show all Records" button on nav

* to update a task from the list, click the update button next to it and change necessary fields and click "Update Button"

* to delete a task from the list, click the delete button next to it
